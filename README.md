# 🧠 Quiz Game Application

# 📌 Project Title
Quiz Game Application

# 📌 Project Overview
The Quiz Game Application is an interactive web-based quiz platform developed using HTML, CSS, and JavaScript(SCT_WD_3). It presents multiple-choice questions to users, evaluates their answers instantly, tracks scores, and displays the final result at the end of the quiz. @SKILLCRAFTTECHNOLOGY
# 🚀 Features


Multiple-choice questions


Instant answer validation


Score tracking


Next question functionality


Final score display


Responsive design


Restart quiz option


# 🛠️ Technologies Used


HTML5

JavaScript (ES6)

# 👤 User Interface
The Quiz Game Application provides a clean and user-friendly interface that allows users to:
View quiz questions one at a time.
Select answers through interactive buttons.
Receive instant feedback on selected answers.
Navigate easily to the next question.
View the final score after completing the quiz.
Restart the quiz without refreshing the page.
Access the application on desktop, tablet, and mobile devices due to its responsive design.
